Netflix Content Analysis
Project Overview

This project analyzes Netflix content data to identify trends in movies and TV shows using SQL and Power BI. The goal of the analysis is to understand content distribution, release trends, and country-wise availability of titles on Netflix.

Objectives

Analyze Netflix dataset to identify trends in content production.

Understand the distribution of Movies vs TV Shows.

Identify top countries producing Netflix content.

Examine yearly growth of content on the platform.

Data Cleaning & Preparation

Cleaned and transformed the dataset using SQL.

Removed missing and inconsistent values.

Structured the dataset for better analysis and visualization.

Dashboard & Visualization

An interactive Power BI dashboard was created to present insights such as:

Total number of titles on Netflix

Distribution of Movies vs TV Shows

Content distribution by genre

Top contributing countries

Year-wise content release trends

Filters and slicers were implemented to allow dynamic exploration of the data by year, genre, and content type.

Key Insights

Movies represent the majority of Netflix content compared to TV shows.

Significant growth in content releases occurred after 2015.

Certain countries contribute the highest number of titles to the Netflix library.

Tools & Technologies

SQL – Data cleaning and querying

Power BI – Data visualization and dashboard creation

Excel / CSV Dataset – Data source
